・はじめに
  BiSHクソゲーは清掃員による清掃員のためのフリーソフトです。

  主な特長：
  1)クソゲー
  2)クソゲー
  3)クソゲー


・ファイル構成
　bishlive.exe    (BiSHクソゲー本体)
　readme.txt　　　(この説明ファイル)


・起動方法
  bishlive.exeをダブルクリックして開いて下さい。


・アンインストール方法
  削除したくなった場合は、フォルダをそのまま削除してください。
  
  
・操作方法
  - マウスを清掃員の上で動かす
    清掃員をジャンプさせる
    
  - 左クリック
    メンバーの必殺技を発動
    （メンバー1人につき1回のみ発動可能）
    
  - F11
    フルスクリーン/ウインドウモード切り替え
    
  - Escキー
    ゲーム終了
