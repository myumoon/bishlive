# bishlive
BiSHクソゲー
====

ライブ中に空からウ◯コが降ってくるクソゲー

## Description

BiSHクソゲーは清掃員による清掃員のためのフリーソフトです。

主な特長：
  1)クソゲー
  2)クソゲー
  3)クソゲー
  
BiSHに申し訳ないくらいのクソゲー

## Requirement

Unreal Engine 4.16.3

## Usage

・マウスを清掃員に乗せる
  清掃員をジャンプさせる
  
・左クリック
  メンバー必殺技
  各メンバー1回ずつ使用可能

・F11
  フルスクリーン
  
・Esc
  終了
